# HIKE LA

#### Overview

##### Explore hikes in the LA area, add hikes you've completed to your profile, and review hikes you've been on.

##### Check out the app [here](https://hike-la-app.herokuapp.com/)

#### Technologies Used

+ Ruby on Rails
+ HTML5
+ CSS3
+ Bootstrap
+ Font Awesome 
+ Google Maps API
